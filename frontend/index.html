Customize channels

Edit the channels, then Save as a named preset (saving with a custom preset's own name updates it). Channels are NFM or AM. Built-in presets can't be overwritten — save under a new name.

Steps through the channels and stops on a transmission until it goes quiet. Set a Priority channel (e.g. Marine Ch 16) to jump to it whenever it's active. Marine VHF (~156 MHz) needs a VHF/marine antenna.

My location

Sets the reference point for distance/bearing. Defaults to central Stockholm — change it to where your antenna is.

ADS-B

starting…

Route and tail number aren't in the ADS-B signal — this queries adsbdb.com over the internet (by callsign / ICAO). Off keeps ADS-B fully offline.

1090 MHz needs a good antenna (ideally a 1090 LNA via Bias-T).

DAB

tuning…

DAB+ ~174–240 MHz. One block carries many stations; pick a station on the right. Stockholm: blocks 12A, 12C, 12D (12C = Sveriges Radio).

AIS

starting…

Click a vessel to add a note. Names & notes are cached, so known ships show up named next time. 162 MHz marine band — data is NOT shared online; needs a VHF/marine antenna.

APRS

starting…

144.800 MHz packet (EU). Needs direwolf installed (brew install direwolf). Plots stations heard direct or via digipeaters; beacons are infrequent, so give it time.

APT (weather sat)

waiting for a pass…

137 MHz needs a satellite pass overhead and a 137 MHz antenna (dipole in a horizontal "V"). Image builds top-down at 2 lines/s over ~10–15 min. Record IQ to decode again later in Replay.

SSTV (image)

waiting for a transmission…

Slow-scan TV pictures. Default is the 2 m calling frequency 144.500 MHz (NBFM). The mode (Martin / Scottie) is auto-detected from the VIS header — a picture builds top-down over ~1–2 min. For HF SSTV (e.g. 14.230 MHz) open Radio, switch to USB, tune, and turn on the SSTV toggle. Record IQ to decode again in Replay.

Pager (POCSAG/FLEX)

starting…

Pager messages (POCSAG 512/1200/2400 + FLEX). Needs multimon-ng (brew install multimon-ng). Traffic and legality vary by region — DAPNET (439.9875) is the amateur network.

ACARS

starting…

~131 MHz aircraft data (EU: 131.725/131.525/131.825). Needs acarsdec built from source (see README). Messages are short and infrequent — leave it running near an airport.

ISM

starting…

ISM band sensors — weather stations, soil/pool sensors, TPMS tyre pressure, remotes, meters. Needs rtl_433 (brew install rtl_433). 433.92 (EU) is busiest; 868.3 (EU) and 315/915 (Americas) carry more. Devices beacon periodically — give it a minute, busiest in the evening.
